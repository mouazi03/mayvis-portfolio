Overview

This desktop GUI application was created using Python's built-in tkinter library. It focuses on building a responsive and user-friendly form interface for basic user input. The layout adapts to resizing and includes entry widgets, radio buttons, checkboxes, a combo box, and functional buttons for reset, exit, and data confirmation.

Key Features

  • Form includes full name entry, residency options, academic program choices
  • ComboBox for program selection (AI, Gaming, Health, Software)
  • Checkboxes for course registration (Programming I, Web Design, Software Engineering)
  • Responsive layout using Frames with padding and relief
  • Reset, Exit, and OK buttons trigger appropriate tkinter events
  • Application opens in a usable, pre-filled state and adapts to window resizing

Built fully in tkinter without any external libraries, this application demonstrates UI fundamentals and interactive design suitable for beginner-level GUI projects.
