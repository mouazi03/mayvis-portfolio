Overview

This project combines insights from multiple labs to provide a comprehensive view of hospital workforce distribution, departmental performance, and resource utilization using Power BI. All dashboards were developed using the same dataset: Hospital_Data.xlsx.

Key Features

  • Dashboard 1: Workforce Overview — gender, age, job roles, department size
  • Dashboard 2: Department Allocation — workload comparison, bed availability, shift analysis
  • Dashboard 3: Resource Utilization — staff-to-patient ratio, role demand, departmental priorities

These visualizations were created by integrating multiple lab requirements and combining advanced analytics tools like decomposition trees, key influencers, and performance analyzers to optimize healthcare operations.
